# Py-Smart-Calc
A robust command-line calculator built to perform arithmetic and statistical operations. This project demonstrates the fundamentals of Python programming, including function definition, control flow, and user input handling.

# üìñ About
This project is a CLI (Command Line Interface) application that prompts the user to input two numbers and select a mathematical operation. It is designed to be a quick and efficient tool for basic calculations.

Supported Operations:
1. Addition: Sum of two numbers.
2. Subtraction: Difference between two numbers.
3. Multiplication: Product of two numbers.
4. Division: Quotient of two numbers.
5. Average: Calculates the mean of two inputs.
6. Percentage: Calculates percentage based on a cumulative total of 200 (useful for exam scoring).

# üõ†Ô∏è Concepts Used
The code implements several core programming concepts:
* Functions (def): Modular code blocks for each operation (add, sub, multiply, etc.) to keep the logic clean and reusable.
* Control Flow (if-elif-else): A decision-making ladder used to execute the specific function corresponding to the user's choice.
* Type Casting: Converting raw user input (strings) into integers using int() for mathematical processing.
